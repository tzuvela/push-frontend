# Push Frontend

Minimal frontend for testing Web Push Notifications
Registers a service worker, requests notification permission, subscribes the user to push notifications, and sends the subscription to the backend

## Features

Service worker registration
VAPID public key retrieval from backend
Push subscription via Push API
Simple UI with a single “Enable Notifications” button

## Backend

This frontend works with the companion backend deployed on Render.

## Deployment

Deployed on Netlify as a static site.
